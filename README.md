Project Name: My Secret Diary 💌
It’s a private, password-protected digital diary where you can write entries, save them, and have them automatically load when you return.
I wanted to create a fun, cute, and secure way for people (or myself) to write down what's on their mind.
I used HTML for the structure, CSS for the  styling, and JavaScript to handle saving entries with localStorage, checking passwords, and displaying messages.
I struggled with making the password system work and storing data so it persists after refreshing the page. I learned how to use localStorage.
